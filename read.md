# 🛡️ Vision-Sentinel-Agent (视觉哨兵智能体)

一个基于大语言模型（LLM）与边缘计算硬件的**软硬件一体化自主追踪人脸识别智能体**。系统不仅具备高精度的面部识别与傲娇/严厉的双重人格语音交互能力，更拥有物理实体的“追踪之眼”，能够根据陌生人定位自主旋转云台进行锁定。

---

## 🌟 核心特性 (Features)

- **⚡ 硬件加速推理**：完美适配 AMD Radeon 780M 等现代强劲核显（基于 DirectML），实现高帧率、低延迟的本地人脸边缘计算。
- **👁️ 物理动态追踪（规划中）**：结合二维度舵机云台，利用人脸框中心点偏离算法，实现摄像头的自主旋转与目标视觉锁定。
- **🧠 大模型动态交互**：接入通义千问大模型（Qwen），根据时间、身份（主人/陌生人）及自定义语音指令，生成极具个性的傲娇或守卫警告文本。
- **🗣️ 双向语音全双工**：支持无线麦克风多轮语音唤醒（呼叫“哨兵哨兵”），并基于微软 Edge-TTS 实现高拟真度的语音反馈。
- **🚨 异步多级报警**：发现未知入侵者时，触发本地带水印存证、图床无感上传、钉钉机器人富文本（Markdown）卡片秒级推送。
- **📊 数据安全与周报**：本地 CSV 自动记录到访日志，支持通过语音指令让 AI 自动汇总生成“近一周安全事件分析简报”。

---

## 🛠️ 系统架构 (Architecture)

### 1. 软件技术栈
- **视觉核心**：`InsightFace` (Antelopev2 模型) + `OpenCV`
- **推理后端**：`ONNX Runtime (DirectML)`
- **大模型驱动**：`OpenAI SDK` + `通义千问 API`
- **语音系统**：`SpeechRecognition` + `Edge-TTS` + `Pygame`

### 2. 硬件技术栈 
- **主控单元**：PC端（负责大模型与视觉推理） + `Arduino / ESP32`（负责微控制器舵机驱动）
- **执行机构**：二维度舵机云台（SG90 或 MG90S 舵机 * 2）
- **采集传感器**：USB 广角夜视摄像头 / 麦克风阵列
